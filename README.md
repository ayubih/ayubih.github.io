# ayubih.github.io

This is the source code for my personal website/blog, built with [Hugo](https://gohugo.io/).

## Local Development

1.  Clone the repository:
    ```bash
    git clone --recursive https://github.com/ayubih/ayubih.github.io.git
    ```
2.  Run the server:
    ```bash
    hugo server
    ```

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions.
